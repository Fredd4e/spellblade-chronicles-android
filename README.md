# Spellblade Chronicles - Android RPG

A story-driven fantasy RPG for Android (and any device with a modern browser).

**Play instantly in your browser!** No installation needed for the web version.

## How to Play on Your Phone (Android)

### Easiest Way (Recommended - No Build Needed)
1. Go to the GitHub Pages URL (once enabled): https://fredd4e.github.io/spellblade-chronicles-android/
2. Open the link in Chrome on your Android phone.
3. Tap the menu (3 dots) > **Add to Home screen** to install it like a native app (PWA).
4. Launch from your home screen and enjoy fullscreen experience.

### Full Native Android App (Advanced)
If you want a true .apk:
1. Clone this repo.
2. Open in Android Studio.
3. But note: Current version is web-based for instant play. A native version may be added later.

## Game Features
- Engaging story with NPCs and choices
- Turn-based combat with sword and spells
- Level up system
- Equipment progression (swords, armor)
- Inventory and shop
- Save/Load progress

## Story
You are a budding Spellblade in the village of Eldoria. Dark creatures threaten the peace. Arm yourself, master magic, defeat foes, and uncover the mystery behind the shadows.

Built with HTML5 + Tailwind + vanilla JS for maximum compatibility and easy play on mobile.
